# Subliminal_Generator
A basic python program for generating a video of subliminals
# How to run
1) Install python on your machine. You can do this via cmd.exe by typing python.exe, and following windows popup. If you're using linux, you know how to do this.
2) Add pillow and opencv-python. You can do this by using the commands:
     pip install pillow
     pip install opencv-python
3) Download these files. You can clone them if you want to make changes here, or just download.
4) Copy the text location of the file these are downloaded into, including Main.
5) Run the command cd (and right click to copy the location) to navigate to the folder.
6) Run python.exe "Caption generator.py"
7) A few questions will pop up, to customize the parameters of the subliminals. (You can edit the Center script.txt and script.txt files to change what text appears in the center or periphery respectively)
8) Wait, and Caption video.mp4 will be created!
